# ZenTech-Azure
Geliştirdiğim Azure API'sini kullanan Firebase tabanlı chatbot sitesi, kullanıcı hesaplarının yetki seviyesine göre kısıtlanmış yapay zekalara erişim sunar. Kullanıcılar, yalnızca görev alanlarına uygun yapay zeka modellerini görebilir. Bu sayede, sistem güvenliği korunurken, kişiye özel yapay zeka çözümleri sağlanır.
